# NeoMutt Cheatsheet


## Navigation (Index)

| Key      | Action                  |
|----------|-------------------------|
| `j` / `k` | next / prev message    |
| `J` / `K` | next / prev mailbox (sidebar) |
| `gg`     | first message           |
| `G`      | last message            |
| `x` / `X` | next / prev page       |
| `Ctrl+N` | next unread             |
| `Ctrl+P` | prev unread             |
| `L`      | open mailbox (sidebar)  |
| `B`      | toggle sidebar          |

## Goto (Index)

| Key  | Action       |
|------|--------------|
| `gi` | Inbox        |
| `gs` | Sent         |
| `gd` | Drafts       |

## Account Switching

| Key     | Account    |
|---------|------------|
| `Alt+1` | gmail1     |
| `Alt+2` | gmail2     |
| `Alt+3` | michigan   |

## Reading (Pager)

| Key  | Action                     |
|------|----------------------------|
| `j` / `k` | scroll line down / up |
| `J` / `K` | scroll line down / up  |
| `x` / `X` | next / prev page       |
| `h`  | back to index              |
| `H`  | toggle full headers        |
| `l`  | view attachments           |

## Actions (Index / Pager)

| Key        | Action                   |
|------------|--------------------------|
| `m`        | compose new message      |
| `r`        | reply                    |
| `R`        | reply-all                |
| `f`        | forward                  |
| `d`        | delete                   |
| `u`        | undelete                 |
| `s`        | save to mailbox          |
| `c`        | change mailbox           |
| `I`        | fetch IMAP mail          |
| `A`        | mark all new as read (index) / add sender to abook (pager) |
| `Z`        | save and quit            |
| `Ctrl+B`   | open abook               |
| `Ctrl+U`   | open URLs with urlscan   |
| `Tab`      | abook query (composer)   |

## Threads (Index)

| Key  | Action           |
|------|------------------|
| `-`  | collapse thread  |
| `_`  | collapse all     |
